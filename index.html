HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages.

Elements

  • <head> 
  • <body> 
  • <title> 
  • <div> 
  • <header>

Tips

  1. Always close your HTML tags.
  2. Use HTML comments wisely.
  3. If possible, use relative URLs.
  4. Use tables for tabular data, CSS for layout.
    • Tables have traditionally been used to lay out content on the page.
    • This was never their intended use.
